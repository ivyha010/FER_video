Facial Expression Recognition from videos using FER library

1. Run  main_emo_detection_video.py 
   

For example: 

Input: "./video_sample_1.mp4"

Output:    Human Emotions  Emotion Value from the Video
0          Angry                         28.37
1        Disgust                          0.06
2           Fear                         23.30
3          Happy                         66.90
4            Sad                         30.89
5       Surprise                         49.22
6        Neutral                         34.95


2. Check the output video named "video_sample_1_output.mp4" in the folder "./output" for an illustration of the output.
